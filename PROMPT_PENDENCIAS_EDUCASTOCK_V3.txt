Criterios de aceite:
- Para um produto com multiplos lotes, o sistema sempre sugere o menor daysToExpiry positivo.
- Se usuario escolher lote diferente, sistema exige justificativa e grava audit log.


3) Push notifications e centro de eventos

- Integrar navegacao ao tocar na notificacao (abrir Alertas).
Criterios de aceite:
- Usuario recebe push de alerta critico.
- Toque na push abre tela de alertas.

4) Cloud Functions (server-side)
- Criar pasta functions com setup TypeScript.
